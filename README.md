# University Campus Network Infrastructure 🌐
> Multi-Campus Network Design & Implementation  
> Simulated using Cisco Packet Tracer

## 📌 About
A full enterprise-grade university campus network infrastructure 
designed and simulated for a multi-location institution spanning:
- 🏢 **Main Campus** — 3 Buildings (Admin, Academic, IT & Labs)
- 🏫 **Smaller Campus** — AI & Machine Learning departments
- 🏥 **Campus Clinic** — Dedicated healthcare network segment

## 🌐 Network Features
| Technology | Details |
|---|---|
| **Topology** | Hierarchical model — Access, Distribution, Core layers |
| **VLANs** | Dept. segmentation: Management, HR, Finance, CS, IT, Labs |
| **Routing** | OSPF · EIGRP · BGP (inter-campus & external) |
| **IP Management** | DHCP for automatic address allocation |
| **Security** | Switchport security · VLAN isolation |
| **Services** | DNS Server · Web Server · Cloud Email Server |

## 👤 My Role — Group Leader
